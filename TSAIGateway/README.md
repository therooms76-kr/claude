# TS AI Gateway

Enterprise AI Management Platform

## 개요

TS AI Gateway는 통합 AI 모델 관리, 프롬프트 최적화, 보안 및 비용 관리를 위한 올인원 솔루션입니다.

## 주요 기능

### 1. 사용자 관리
- 개인 프로필 관리
- 프롬프트 템플릿 관리
- 토큰 사용량 추적/한도
- 대화 히스토리 관리
- 그룹 채팅
- 실시간 번역

### 2. 인증 및 보안
- OAuth 기반 소셜 로그인 (Google, GitHub)
- API 키 관리
- 보안 이벤트 로깅
- 파일 업로드/버전 관리
- 민감정보 필터 (Presidio SDK)

### 3. 프로젝트 관리
- 프로젝트 생성 및 설정
- 프로젝트 태그/메타데이터 관리

### 4. AI 모델 통합
- 지원 모델 카탈로그
- 호환 API 및 어댑터

### 5. 프롬프트 관리
- 산업별 기본 템플릿 (200개 이상)
- Git 기반 프롬프트 버전 관리

### 6. 대화 관리
- 대화 생성/컨텍스트 유지
- 다양한 메시지 역할과 타입
- SSE 기반 스트리밍 응답
- 대화 북마크 및 태그

### 7. 크레딧 시스템
- 서비스 크레딧 시스템
- 신규 사용자 환영 크레딧 (30일 유효)
- 사용량 한도 및 알림

### 8. 통계/모니터링
- 실시간 사용량 대시보드
- 비용 분석 및 예측

### 9. 결제 시스템
- 종량제 결제
- 다양한 결제 수단 (신용카드, 카카오페이, 네이버페이)

## 프로젝트 구조

```
TSAIGateway/
├── index.html          # 메인 HTML 파일
├── css/
│   └── styles.css      # 스타일시트 (짙은 푸른색 계열)
├── js/
│   └── main.js         # JavaScript 인터랙션
├── images/             # 이미지 리소스
└── README.md          # 프로젝트 문서
```

## 디자인 특징

- **컬러 스킴**: 짙은 푸른색 계열 (Deep Blue)
  - Primary Blue: #0a2540
  - Secondary Blue: #1a3a5c
  - Accent Blue: #2563eb
  - Light Blue: #3b82f6

- **반응형 디자인**: 모바일, 태블릿, 데스크톱 지원
- **애니메이션**: 스크롤 기반 페이드인 효과
- **인터랙션**: 버튼 리플 효과, 부드러운 스크롤

## 사용 방법

1. 웹 브라우저에서 `index.html` 파일을 엽니다.
2. 네비게이션 메뉴를 통해 각 섹션으로 이동할 수 있습니다.
3. 각 기능 카드를 클릭하여 상세 정보를 확인할 수 있습니다.

## 개발 단계

- **Phase 1**: 기본 기능 (대부분의 핵심 기능)
- **Phase 2**: 고급 기능 (파일 버전 관리, 민감정보 필터)

## 기술 스택

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Intersection Observer API
- CSS Animations
