상세 기능

사용자 관리

개인 프로필 관리

표시 이름, 선호 언어, 시간대 등 설정 및 선호 AI 모델, 관심사 등을 통한 사용자 패턴 분석 및 추천 기능 제공

Phase 1

프롬프트 템플릿 관리

법률, 마케팅 등 업무 분야별 최적화된 기본 템플릿 제공 및 사용자의 커스텀 템플릿 저장, 카테고리 분류, 사용 통계 확인 기능

Phase 1

토큰 사용량 추적/한도

실시간 input/output 토큰 사용량 계산 및 기록. 일별/주별/월별, 프로젝트별, 모델별 분석 제공 및 관리자/팀 리더의 토큰 사용 한도 설정 및 알림/차단 기능

Phase 1

대화 히스토리 관리

모든 대화 안전하게 저장, 지능적인 검색, 분류, 태그 부여 및 AI를 활용한 긴 대화 자동 요약 기능 제공

Phase 1

그룹 채팅

여러 사용자가 하나의 대화에 동시 참여하여 AI와 소통 및 컨텍스트 공유. 실시간 동기화 (WebSocket) 및 히스토리 공유

Phase 1

실시간 번역

다국어 팀을 위해 각 사용자의 선호 언어로 메시지를 자동 번역하여 언어 장벽 제거. 원문 보기 및 예외 목록 설정 가능

Phase 1

인증 및 보안

OAuth 기반 소셜 로그인

Google, GitHub 등 OAuth 프로토콜을 통한 간편 로그인 지원 (카카오, 네이버 추가 예정)

Phase 1

API 키 관리

개발자 및 고급 사용자가 프로그래밍 방식으로 Gateway에 접근할 수 있도록 API 키 발급 및 관리

Phase 1

보안 이벤트 로깅

로그인 시도/실패, 권한 변경 등 모든 보안 관련 이벤트를 타임스탬프, IP와 함께 상세 기록 및 비정상 패턴 자동 탐지/경고 기능 제공

Phase 1

파일 업로드/버전 관리

AI와의 대화에서 참고할 문서/이미지 등을 업로드하고, Git과 유사한 버전 컨트롤 기능을 통해 파일 버전(1.0, 1.1 등)을 추적 관리

Phase 2

민감정보 필터

Presidio SDK를 활용하여 텍스트/이미지 내 개인식별정보(PII)를 식별하고 마스킹, 대체 등의 연산자를 적용해 비식별화를 지원

Phase 2

프로젝트 관리

프로젝트 생성 및 설정

프로젝트 이름, 기본 AI 모델, 기본 시스템 프롬프트 설정. 비공개, 팀 공유, 조직 공개 등 가시성 및 접근 권한 설정

Phase 1

프로젝트 태그/메타데이터

프로젝트 분류 및 검색용 태그 부여. 담당자, 우선순위, 상태 (진행 중, 완료 등) 등의 메타데이터 관리

Phase 1

AI 모델 통합

지원 모델 카탈로그

사용 가능한 모든 AI 모델의 상세 정보(컨텍스트 크기, 지원 기능, 입력/출력 토큰당 가격)를 투명하게 제공

Phase 1

호환 API 및 어댑터

각 AI 제공자의 API 형식을 추상화하여 통일된 인터페이스 제공. OpenAI 호환 API 엔드포인트 지원으로 마이그레이션 용이성 확보

Phase 1

프롬프트 관리

산업별 기본 템플릿

법률, 마케팅, 소프트웨어 개발 등 다양한 분야를 위한 200개 이상의 검증된 프롬프트 템플릿 제공

Phase 1

프롬프트 버전 관리

프롬프트에 Git 유사 버전 관리 시스템을 적용하여 수정 이력 추적(커밋), 브랜치를 통한 여러 버전 동시 실험 및 성능 메트릭 기록

Phase 1

대화 관리

대화 생성/컨텍스트 유지

대화별 독립적인 컨텍스트 유지. "이전 대화 참조" 기능을 통한 컨텍스트 연결 가능

Phase 1

메시지 역할과 타입

메시지에 system, user, assistant 역할 부여. 함수 호출 시 tool-call, tool-result 타입 사용. 이미지/PDF 등 다양한 첨부물 포함 가능

Phase 1

스트리밍 응답

SSE 프로토콜을 사용한 실시간 텍스트 스트리밍으로 사용자 경험 향상 및 응답 중단 기능 제공

Phase 1

대화 북마크 및 태그

중요한 대화에 북마크 설정 및 태그 부여. 고급 검색 기능을 통해 방대한 기록에서 정보 검색

Phase 1

크레딧

서비스 크레딧 시스템

서비스 크레딧 운영(프로모션, 신규 가입자 등)

Phase 1

환영 크레딧

신규 사용자에게 무료 체험용 크레딧 부여 (30일 유효기간)

Phase 1

사용량 한도 및 알림

사용자/프로젝트/조직 단위 토큰 한도 설정 및 임계치 도달 시 경고/차단 알림 발송

Phase 1

통계/모니터링

사용량 대시보드

총 토큰 사용량, 비용, 활성 사용자 수 등 핵심 지표 요약. 시간별 사용량, 모델별 사용 비율, 프로젝트별 통계 제공

Phase 1

비용 분석 및 예측

일별/주별/월별 비용 추이 분석 및 향후 비용 예측. 모델 단가표 명시

Phase 1

결제 시스템

종량제 결제

사용량 기반 비용 지불 모델. 서비스 토큰 구매 및 자동 충전 기능, 투명한 사용 내역 기록 및 청구서 생성

Phase 1

결제 수단

신용카드(Visa, Mastercard), 카카오페이/네이버페이 등 간편결제 및 기업 고객을 위한 청구서 결제 지원

Phase 1
